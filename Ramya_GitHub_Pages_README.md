# Ramya K — Internship Mission Plan 2026

A complete internship preparation website covering everything from resume fixes to day-by-day task calendars.

## How to Deploy on GitHub Pages (Free)

### Step 1 — Create a GitHub account
Go to [github.com](https://github.com) and sign up if you haven't already.

### Step 2 — Create a new repository
- Click the **+** icon → **New repository**
- Name it: `internship-plan-2026` (or any name)
- Set to **Public**
- Click **Create repository**

### Step 3 — Upload the files
- Click **uploading an existing file**
- Drag and drop `index.html` and this `README.md`
- Click **Commit changes**

### Step 4 — Enable GitHub Pages
- Go to repository **Settings** → **Pages** (left sidebar)
- Under **Source**, select **Deploy from a branch**
- Branch: **main**, Folder: **/ (root)**
- Click **Save**

### Step 5 — Get your URL
- Wait 2-3 minutes
- Your site will be live at: `https://[your-github-username].github.io/internship-plan-2026`
- Share this URL anywhere!

## What's Inside the Website

- **Today Tab** — Day 1 exact action list with timestamps
- **Profile Tab** — Honest assessment of strengths and gaps
- **Calendar Tab** — Week-by-week + day-by-day task breakdown (Weeks 1-20)
- **Companies Tab** — All target companies with fit analysis
- **Resume Fix Tab** — Every specific issue with exact replacement text
- **DSA Plan Tab** — Topic order, milestones, daily routine
- **Behavioral Tab** — All 16 Amazon LPs with story sources, DM templates
- **Resources Tab** — All platforms with priority and timing
